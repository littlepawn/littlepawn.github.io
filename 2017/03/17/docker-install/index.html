    4. 容器相关命令

      docker ps -a 查看容器
      docker start [container] 开启容器
      docker attach [container] 进入一个启动的容器
      docker stop [container] 停止容器
      在容器里：ctrl+p ctrl+q退出但不停止
      ctrl+d关闭

    5. docker和宿主机之间的相互传输

      从docker到宿主机：docker cp [containerID]: /xx /xx
      从宿主机到docker：docker inspect -f
